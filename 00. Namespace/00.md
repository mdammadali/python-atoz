## types of namespaces:
    - 01. Built-In
    - 02. Global
    - 03. Local
    - 04. Enclosing or nonlocal

01: The Built-in Namespace
```
>>> dir(__builtins__)
['__builtins__', '__doc__', '__loader__', '__name__', '__package__', '__spec__']
```

02: The Global Namespace
```
>>> dir()
['__annotations__', '__builtins__', '__doc__', '__loader__', '__name__', '__package__', '__spec__', 'numbers']
```
node: number last insert in global scope

03: The Local Namespace

```
global_variable = "global"
def outer_func():
    # Nonlocal scope
    nonlocal_variable = "nonlocal"
    def inner_func():
        # Local scope
        local_variable = "local"
        print("Inner Variable:", local_variable)
        print("Outer Variable:", nonlocal_variable)
        print("Global Variable:", global_variable)
    inner_func()
outer_func()
```
